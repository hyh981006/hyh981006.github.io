Unity Input类知识点积累

发表于2021-04-04|更新于2021-05-05|Unity3d基础知识
|字数总计:667|阅读时长:2分钟|阅读量:

属性

  • anyKey，按住任意键
  • anyKeyDown，按下任意键
  • gyro，返回默认陀螺仪
  • inputString，返回当前帧键盘的输入字符
  • mousePosition，鼠标坐标
  • multiTouchEnable，是否支持多点触控
  • touchCount，当前所有的触摸状态列表长度，
  • touchSupport，返回当前运行应用的设备是否支持触控

    方法

  • GetAxis，根据名称获取虚拟轴的值
  • GetButton，GetButtonDown，GetButtonUp，虚拟键
  • GetKey，GetKeyDown，GetKeyUp，键盘
  • GetMouseButton，GetMouseButtonDown，GetMouseButtonUp，鼠标
  • GetTouch 触屏

    鼠标相关

  • GetAxis，根据名称获取虚拟轴的值
    float x=Input.getAxis(“Mouse X”);
    float y=Input.getAxis(“Mouse Y”);
    float scroll=Input.getAxis(“Mouse ScrollWheel”);
  • GetMouseButton，鼠标
  • GetMouseButtonDown，鼠标按下
  • GetMouseButtonUp，鼠标抬起
  • mousePosition，鼠标坐标

    键盘相关

  • GetAxis，根据名称获取虚拟轴的值
    float x=Input.getAxis(“Horizontal”);
    float y=Input.getAxis(“Vertical”);
  • GetKey，
  • GetKeyDown
  • GetKeyUp

    虚拟键相关

  • GetButton，
  • GetButtonDown，
  • GetButtonUp

触屏相关

  • touchCount，当前所有的触摸状态列表长度，
  • GetTouch 触屏
  • touches 触屏信息列表
  • multiTouchEnable，是否支持多点触控
  • touchSupport，返回当前运行应用的设备是否支持触控

    touch结构体的内容

    属性
  • fingerId, 返回触摸的索引
  • Position，触摸点坐标，返回Vector2
  • deltaPosition 触摸位置改变量，返回Vector2
  • deltaTime 距离上次触摸位置改变的时间，返回float
  • tapCount 单击次数，返回int
  • phase 触摸状态，返回触摸状体（枚举类型，具体看下页）；
    触摸状态
  • began 刚开始触屏，
  • Moved 移动
  • stationary 触屏后一直没动
  • Ended 手指离开屏幕
  • Canceled 系统取消了触摸，当触点过多的时候则该状态
